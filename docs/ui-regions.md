# UI 區域命名（溝通用詞）

定義各 UI 區域的中英文名稱，方便日後溝通。原則：**設定區沿用畫面既有標籤；分組區 + 預覽控制才另造名**。

```
┌──────────────┬───────────────────┬───────────────┐
│  ① 設定區     │   ② 分組區          │   ③ 預覽區     │
│   Config     │   Grouping        │   Preview     │
│              │ ┌───────────────┐ │               │
│ 範本 / 圖片   │ │ⓐ 模式切換 Mode │ │  表格/投影片   │
│ 配置 / 輸出   │ │ⓕ 檔案列        │ │  預覽          │
│              │ ├───────────────┤ │               │
│              │ │ⓑ Group 區(綠) │ │ ┌───────────┐ │
│              │ │ⓒ Index 區(藍) │ │ │ⓔ 預覽控制  │ │
│              │ ├───────────────┤ │ │ 填法/格高  │ │
│              │ │ⓓ 卡片區 Cards │ │ └───────────┘ │
│              │ └───────────────┘ │               │
└──────────────┴───────────────────┴───────────────┘
```

## ① 設定區 Config（左欄）

內部區塊隨模式不同；**用畫面既有標題**即可，不另造名。

| 中文（畫面標籤） | English | 出現於 |
|---|---|---|
| 範本 | Template | 共同 |
| 圖片資料夾 | Images | 依檔名 |
| SN 欄 | SN column | 依SN清單 |
| SN 清單來源 | SN source | 依SN清單 |
| 資料來源 | Sources | 依SN清單 |
| 每頁網格 / Excel 配置（分隔符·字體·起始 cell·圖片填法） | Grid / Config | 依檔名 |
| 輸出檔名 + 匯出鈕 | Output / Export | 共同 |

## ② 分組區 Grouping（中欄）

畫面沒有標題，**另造名**。

| 代號 | 中文 | English | 是什麼 |
|---|---|---|---|
| ⓐ | 模式切換 | Mode | 依檔名 / 依 SN 清單 |
| ⓕ | 檔案列 | File bar | 「檔案(N)·選 X / 統一·各欄 / 設定裁剪」那條 |
| ⓑ | Group 區 | Group | 綠色總覽（group 標籤） |
| ⓒ | Index 區 | Index | 藍色總覽（index/SN 標籤） |
| ⓓ | 卡片區 | Cards | group/資料夾 橫條卡片 |

## ③ 預覽區 Preview（右欄）

| 代號 | 中文 | English | 是什麼 |
|---|---|---|---|
| — | 預覽 | Preview | 表格 / 投影片 |
| ⓔ | 預覽控制 | Controls | 圖片填法、填滿格高、換頁、縮放 |

## 配色語意（跨區一致）

- **Group = 綠**、**Index = 藍**（軸的代表色，分組區）
- **依範本 = 灰**（範本鎖定的標籤格，預覽區）
