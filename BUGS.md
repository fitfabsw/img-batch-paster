# Bug Tracker

未當場修掉的 bug 都記在這裡。修好後把狀態改 `Fixed` 並附 commit hash（不刪除，保留歷史）。

狀態：`Open`（待修）/ `Investigating`（查不到根因）/ `Fixed`（已修）/ `Wontfix`（不修，附原因）

格式：

```
## [B-NNN] 一句話標題
- **狀態**: Open
- **發現**: YYYY-MM-DD
- **重現**: 步驟…
- **預期 / 實際**: …
- **備註 / 線索**: …
- **修正**: （Fixed 時填 commit hash + 一句說明）
```
