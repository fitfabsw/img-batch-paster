# Bug Tracker

所有 bug 都記在這裡（含已修），不刪除、保留歷史。純 UX/feature/refactor 不算 bug。

狀態：`Open`（待修）/ `Investigating`（查不到根因）/ `Fixed`（已修，附 commit）/ `Wontfix`（不修，附原因）

格式：`[B-NNN] 標題` + 狀態 / 發現 / 重現 / 預期·實際 / 備註 / 修正。

> 註：B-001 之後為 2026-06-27 回溯補登，多為從 git 歷史重建，日期為近似、重現步驟為摘要。
